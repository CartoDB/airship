# Lists

Airship provides utility classes to specify the typography look and feel. You can use this clases in any kind of HtmlElement.

### Bullet list

```html
  <h1 class="as-subheader">This is a title</h1>
  <p class="as-body">
    I'm thinking two circus clowns dancing. You? Pretend. You pretend the feelings are there, for the world, for the people around you. Who knows? Maybe one day they will be. I'm really more an apartment person.
  </p>
  <ul class="as-list">
    <li class="as-list__item">I'm thinking two circus clowns dancing. You? Pretend.</li>
    <li class="as-list__item">You pretend the feelings are there, for the world, for the people around you.</li>
    <li class="as-list__item">Who knows? Maybe one day they will be.</li>
    <li class="as-list__item">I'm really more an apartment person.</li>
  </ul>
```

# Breadcrumb

A breadcrumb is used to show hierarchy between content.

```html
showSource: true
---
<ul class="as-breadcrumb">
  <li class="as-breadcrumb__item">
    <a href="#">Home</a>
  </li>
  <li class="as-breadcrumb__item">
    <a href="#">Users</a>
  </li>
  <li class="as-breadcrumb__item">
    New user
  </li>
</ul>
```

## CSS Variables

```
.as-breadcrumb {
  --as--breadcrumb--color--current
  --as--breadcrumb--color--visited
}
```
